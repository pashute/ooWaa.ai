# humANDai - Frontend Application

ooWaa.ai: The AI reformer from Overstand. 
Knows what it doesn't. Teams with you to get it right.

humANDai is a chat orchestrator that you can rely on. 
It keeps the conversation on track, at a slow human pace, with context, and with you at the wheel. 

It works with the humandbrAIn which maintains a proactive modular comprehension base of knowledge graphs, with your studies, experience, reactions, and persona. 

It plans the conversation with you, 

React Native mobile application built with Expo.
